# Real Estate Listings Scraper for Nehnutelnosti.sk

This project scrapes real estate listings from [nehnutelnosti.sk](https://www.nehnutelnosti.sk) and extracts structured property data using a scikit-learn style `Pipeline`.

Not needed to run! This scraper was used in initial stage of diploma thesis to gather data. 1000 randomly chosen lsitings are saved in Listings.json in database directory.

## Usage

### 1. Install Requirements

```pip install requests beautifulsoup4 pandas scikit-learn```


### 2. Run

Run desired cells
